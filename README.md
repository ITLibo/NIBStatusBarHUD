# NIBStatusBarHUD
一个状态栏显示的HUD

## 显示成功信息
```object
    [NIBStatusBarHUD showSucces:@"加载成功"];
```

## 显示失败信息
```object
     [NIBStatusBarHUD showError:@"加载失败!"];
```
## 显示文字信息
```object
     [NIBStatusBarHUD showMessage:@"显示文字"];
```
## 显示加载信息
```object
     [NIBStatusBarHUD showLoading:@"正在加载..."];
```
## 隐藏
```object
     [NIBStatusBarHUD dismiss];
```
